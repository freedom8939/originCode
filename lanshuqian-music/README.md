# 纯原生js,css,html项目

## 项目介绍：

> 本项目是一款在线的音乐播放器项目，项目由
> HTML5, CSS3, JavaScript实现。
> 项目中使用了两个前端第三方库(均以cdn方式引进):
> + Cookies
> + jquery
> <br/> 
> 
> 后端基于JAVA实现，实现了用户登录,注册,退出登录，JDK版本: JDK11。使用SpringBoot进行快速开发，主要技术栈：
> <br/>
> + Spring + SpringMVC + SpringBoot 框架
> + MyBatis Plus 数据访问框架
> + MySQL8.0 数据库
>
